# `Discovery` Discover the world of microcontrollers through [Rust](https://www.rust-lang.org/)! There are currently two versions of this book.
Both of them provide an introduction to microcontrollers and how to use Rust with them.
The first is older and uses an F3 Discovery circuit board, while the second is newer and uses a micro:bit circuit board instead.

# `发现` 通过[Rust](https://www.rust-lang.org/)发现微控制器的世界！
目前有两个版本的这本书。
它们都介绍了微控制器的基础知识以及如何使用Rust进行开发。
第一个版本较旧，使用F3 Discovery电路板，而第二个版本较新，使用micro:bit电路板。
- 阅读较新的书，使用micro:bit：https://docs.rust-embedded.org/discovery/microbit - 阅读较旧的书，使用F3 discovery电路板：https://docs.rust-embedded.org/discovery/f3discovery - 从这个代码库开始工作示例 - 有问题吗？ - 在我们的[GitHub讨论区](https://github.com/rust-embedded/discovery/discussions)中查看 - 可能已经有人回答了 - 如果没有，开始一个新的讨论 - 发现了问题？ - 在我们的[GitHub问题页面](https://github.
com/rust-embedded/discovery/issues) - Maybe there is already a workaround - If not, please open a new one - or even better - a [pull request](https://github.com/rust-embedded/discovery/pulls) for solving it - Have fun and enjoy!
